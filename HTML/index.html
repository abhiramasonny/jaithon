What is Jaithon?

JAITHON is a programming language carefully put together for teaching children and adults the fundamentals of coding. Its simplicity and intuitive syntax make it an ideal choice for young learners, allowing them to grasp programming concepts easily and enjoyably. The JAI programming Language is perfect for anyone regradless of age. By providing a beginner-friendly environment, The JAITHON Language removes the tech-savvy knowledge needed with traditional programming languages, making it accessible to people with little to no prior coding experience

Visit our Github Page!

What problems are we addressing?

Problem 1 - Coding is hard

To an experienced developer, coding may seem easy. All the lines flow just as you would expect. However, for beginners, this is not the case. Many people don't learn programming because they feel that it is very complex and hard to learn. this is what we are trying to fix with JAITHON. Jaithon has syntax specifically designed to look like English text. We provide the infrastructure necessary to learn a more difficult and practical coding language easier.

Problem 2 - Coding isnt fun

As a developer myself, it pains me to type these lines, but coding isn't fun for everyone. Some people might actually like going outside... and others might not like the idea of spending hours upon hours trying to debug a line of code. With Jaithon however, we may not be able to fix the first problem, but we for sure can fix the second. With our language, any syntax issues will become apparent immediately with our advanced error handling. We have made it so that with our simple syntax it is almost impossible to create a bug that isn't straightforward. This almost solves all of our issues but there is one left, which will be addressed in the next point!

Problem 3 - Why?

Before I explain the 3rd problem we solve, you need to know why? What is the purpose of our language? I've stated before that it is to teach children, but if children wanted to learn, there are 100s of other options to learn. The one thing that makes my language special and different than these 100s options is the syntax. The syntax -or the structure of the language- bears a strong resemblance to Python. With my language, you can almost fluently read and write basic Python code because of how the language is structured.
